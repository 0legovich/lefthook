# Contributing
First off, thanks for taking the time to contribute! Feel free to make Pull Request with you changes.


# Requirements

Go >= 1.13.1
Ruby >= 2.x

# Proccess

1. Fork repo
2. git clone <forked_repo>
3. run commands:
```bash
go build
bundle install
```
4. Make changes
6. Test it
```bash
bundle exec rspec spec/
```
7. Push your changes in repo
